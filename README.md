# 🎬 Every Cinematic Universe

A discovery and exploration site for movies across cinematic universes.  
Live at: [everycinematicuniverse.com](https://everycinematicuniverse.com)

## Stack
- Next.js + TypeScript
- Netlify Functions (OMDB API for posters & release years)
- `universe.ts` — manually curated title lists

## Contributing
See [CONTRIBUTING.md](./CONTRIBUTING.md)  
Want to add a universe or missing titles? Open a PR!

## Setup Locally
```bash
npm install
cp .env.example .env.local  # add your OMDB key
npm run dev
```
