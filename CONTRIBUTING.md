# Contributing to Every Cinematic Universe

## How to Add a Universe or Titles

1. Fork this repo
2. Edit `src/data/universe.ts` — add your titles to the correct universe array
3. Make sure each title matches the OMDB title exactly (for poster fetching to work)
4. Open a PR with the title: `[Universe Name] Add: <Movie Title>`
5. Wait for review — only the maintainer merges

## Rules
- No changes to infrastructure, Netlify functions, or layout without discussion first
- Open an Issue before large changes
- One universe/topic per PR — keep them small and reviewable
